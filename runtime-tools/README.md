Pinned runtime assets for the media resolver.

- `linux-x64/yt-dlp-2025.12.08`: official upstream `yt-dlp` Linux binary
- `linux-x64/deno-v2.6.4-x86_64-unknown-linux-gnu.zip`: official Deno Linux archive used to provide the JavaScript runtime expected by modern `yt-dlp` YouTube extraction

The app can also use explicit local overrides:

- `YT_DLP_PATH`
- `DENO_PATH`
